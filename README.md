# Arrhythmia Detection - Frontend

The user interface for the Arrhythmia Detection system, built with React and Vite.

## 🚀 Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Environment Variables**:
   Create a `.env` file in this directory and add:
   ```env
   VITE_API_BASE_URL=http://localhost:8000
   ```

3. **Run Development Server**:
   ```bash
   npm run dev
   ```

## 🛠️ Tech Stack
- **Framework**: React 18
- **Build Tool**: Vite
- **Styling**: Tailwind CSS
- **Icons**: Lucide React
- **Animations**: Framer Motion

## 📁 Key Directories
- `src/pages`: Main application views (Dashboard, AI Assistant, ECG Upload).
- `src/components`: Reusable UI components.
- `src/context`: Authentication and State management.
